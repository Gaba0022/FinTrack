Projeto: FinTrack CriptoMoedas

Descrição:
Sistema para monitoramento de cotações de criptomoedas, envio de alertas via WhatsApp e armazenamento do histórico de preços para análise e exportação.

Tecnologias:
- Backend:
  • .NET 8
  • Entity Framework Core
  • JWT para autenticação
  • BCrypt para hash de senhas
  • AutoMapper para mapeamento de DTOs
- Banco de Dados:
  • MySQL
- Frontend:
  • React com TypeScript
  • Axios para requisições HTTP
  • CSS para estilização
- Infraestrutura:
  • Docker
  • Twilio API para envio de mensagens via WhatsApp

Entidades Principais:
1. User
   • Id (Guid)
   • Email
   • PasswordHash
   • DisplayName
   • CreatedAt
   • Relacionamento: Alerts (PriceAlert)

2. Crypto
   • Id (string)
   • Symbol
   • Name
   • LastPrice
   • UpdatedAt
   • Relacionamento: Alerts (PriceAlert), History (PriceHistory)

3. PriceAlert
   • Id (Guid)
   • UserId
   • CoinId
   • TargetPrice
   • Direction (Above / Below)
   • IsActive
   • CreatedAt
   • TriggeredAt
   • LastNotifiedAt

4. PriceHistory
   • Id (long)
   • CryptoCoinId
   • Price
   • Timestamp
   • Relacionamento: Crypto

Casos de Uso Funcionais:
1. Monitorar CriptoMoedas
   - Consulta periódica de preços via API CoinGecko
   - Atualização do último preço na tabela Crypto
   - Registro do histórico de preços em PriceHistory

2. Autenticação de Usuário
   - Cadastro com CreateUserDTO (hash de senha via BCrypt)
   - Login com LoginUserDTO (verificação de senha)
   - Geração de JWT para autenticação em endpoints

3. Alertas de Preço
   - Usuário cria/edita alertas com PriceAlert
   - Direção: acima ou abaixo de determinado valor
   - Ativação de alerta quando condição é atendida
   - Notificação via WhatsApp usando Twilio
   - Atualização de status IsActive e LastNotifiedAt

4. Exportação de Histórico
   - Usuário pode exportar histórico de preços de uma criptomoeda em formato CSV
   - Baseado nos registros da tabela PriceHistory

5. Gerenciamento de Usuário
   - Atualização de DisplayName e senha via UpdateUserDTO
   - Garantia de segurança com hash de senha e validação de campos
